## 1. 计量边界

当前系统的 IFRS 16 计量引擎覆盖以下已自动化逻辑：

| 主题 | 系统处理 | 准则映射 | 回归用例 |
|---|---|---|---|
| 后付固定租金 | 将固定租赁付款额按日折现为初始租赁负债，后续按实际利率法确认利息，并按租期折旧 ROU | IFRS 16.22-28, 36-38 | TC-01 |
| 先付固定租金 | commencement 当天或之前支付的先付租金不纳入初始租赁负债，作为已支付租赁付款额计入 ROU | IFRS 16.24, 26-27 | TC-02 |
| 免租期 | 免租期不产生付款冲减，但仍在租期内进行负债利息滚动与 ROU 折旧 | IFRS 16.23-28, 36-38 | TC-04 |
| 变量租金 | sales-based / turnover rent 不进入租赁负债现值，在发生月份费用化 | IFRS 16.38(b) | TC-12 |
| 非租赁成分 | CAM、服务费等非租赁成分不进入租赁负债，在发生月份作为非租赁费用列示 | IFRS 16.12-17 | TC-13 |
| 短期租赁豁免 | `lease_scope=short_term_exempt` 时不确认 ROU/租赁负债，固定租金按直线法费用化 | IFRS 16.5-8 | TC-14 |
| 低价值资产豁免 | `lease_scope=low_value_exempt` 时不确认 ROU/租赁负债，固定租金按直线法费用化 | IFRS 16.5-8 | TC-15 |
| 非租赁合同 | `lease_scope=not_a_lease` 时不进入 IFRS 16 计量，仅保留台账与运营管理信息 | IFRS 16.9-11 | TC-16 |

## 3. 方法说明

初始租赁负债：

- 固定租赁付款额纳入现值计算。
- 变量租金与非租赁成分从现值计算中剔除。
- commencement 当天或之前支付且标记为 `prepaid` 的租赁付款额不形成未来租赁负债。

使用权资产初始计量：

- 初始 ROU = 初始租赁负债 + 初始直接费用 + 先付租金 - 租赁激励 + 复原成本。

后续计量：

- 租赁负债按日实际利率法滚动。
- ROU 按租期日数平均折旧。
- 末期 rounding residual 通过调整项归零，避免尾差残留。

费用化项目：

- `variable` 类型付款列示为变量租金费用。
- `non_lease` 类型付款列示为非租赁成分费用。
